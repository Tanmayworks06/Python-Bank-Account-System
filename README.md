# 🏦 Python Bank Account System

A simple command-line banking system built using Python.

This project allows users to create a bank account, choose account type, deposit money, withdraw money, and view their updated balance.

## 🚀 Features

- Create account with holder name
- Select account type (Saving / Current)
- Add opening balance
- Withdraw money
- Deposit money
- Check insufficient balance
- Show final account details

## 🛠️ Technology Used

- Python

## 📚 Concepts Practiced

- Variables
- User Input
- If-Else Statements
- Arithmetic Operations
- f-strings
- Basic Banking Logic
